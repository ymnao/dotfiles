# 秘密情報ファイルの取り扱い

- このファイルは秘密情報を含む可能性が高い。**内容(値)を返答・ログ・
  コミットメッセージに書かない**。参照が必要なときはキー名のみ言及する
- **git にコミットしない**。`git add` の対象に含めない。gitignore
  されていない場合は追加を提案する
- 内容を外部(API・コマンドの引数経由でのネットワーク送信)に渡さない
- コピー・移動する場合もリポジトリ外またはgitignore 済みの場所のみ
